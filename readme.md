# Шортнер ссылок

Небольшая консольная программа на Java для создания кратких ссылок. Например `suren.com/ABCDEFGH`

## Возможности

- Регистрация и вход по UUID.
- Создание коротких ссылок на произвольные URL
- Ограничение по числу использований.
- Срок действия: от 5 минут и до бесконечности.
- Автоматическая очистка просроченных/исчерпанных ссылок.
- Глобальное использование шорт-ссылки в любой момент: достаточно ввести текст, содержащий `suren.com/[код]` — ссылка будет «израсходована» и открыта в браузере.

## Как пользоваться

После запуска вы увидите главное меню:

1) Войти по UUID — введите ранее выданный UUID.
2) Создать UUID — регистрирует нового пользователя и сразу авторизует.
0) Выход — завершение программы.

После входа открывается меню аккаунта:

UUID: <ваш-uuid>
1) Посмотреть мои ссылки
2) Создать ссылку
3) Удалить ссылку
0) Выйти из аккаунта

### Просмотр и удаление
- Посмотреть мои ссылки — печатает список ваших ссылок с текущей статистикой (использования/лимит, срок действия).
- Удалить ссылку — выбираете номер из списка, ссылка удаляется (только свои ссылки).

## Архитектура и модули

### `Main.java` — консольный интерфейс
- Организует главное и аккаунт-меню, читает ввод.
- Глобальная обработка шорт-ссылки: любой введённый текст проверяется на наличие префикса `suren.com/` и кода.
- Валидация URL (localhost, IPv4, обычные домены с точкой и; без пробелов).
- Выбор срока действия (5/30/60 минут, неделя, месяц или null — без ограничений).
- Генерация уникального кода.
- Открытие целевой ссылки через `Desktop.getDesktop().browse(...)`.

### `MemoryStore.java` — хранилище
- Ведет учет владельцев, исходного URL, короткого кода, количества использований и тд.
- Проверяет валидность.
- Инициализирует генерацию ссылки.
- Удаляет ссылку по необходимости.
- Удаляет ссылку по просрочке.

### `ShortCodeGenerator.java` — генератор кодов
- Гененриует уникальный код.
- Алфавит кодов: `A–Z a–z 0–9` и `-_`

## Формат короткой ссылки

- Префикс домена: `suren.com/`
- Код: 8 символов из `[A-Za-z0-9_-]`
- Пример: `suren.com/Ab9_XY-1`
